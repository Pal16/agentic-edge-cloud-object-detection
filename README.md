## Overview
This project presents a scalable, agentic object detection system that dynamically
routes inference between edge and cloud models to balance accuracy, latency, and cost.

## Key Contributions
- Edge-first real-time inference using YOLOv8
- Transformer-based cloud refinement (RT-DETR)
- Agent-based dynamic routing
- Formal COCO evaluation (AP, AP-S/M/L)
- Statistical significance testing
